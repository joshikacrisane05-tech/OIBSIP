**Task-1: Iris Flower Classification**

**Overview**

This project is an implementation of the **Iris Flower Classification** task using the **K-Nearest Neighbors (KNN)** algorithm in Python. The model classifies iris flowers into one of three species—Setosa, Versicolor, and Virginica—based on their sepal and petal measurements.

**Objective**

To build a machine learning model that predicts the species of an iris flower using the Iris dataset.

**Technologies Used**

* Python
* Scikit-learn

 **Dataset**

The project uses the built-in **Iris dataset** available in the `scikit-learn` library.

**Steps**

1. Load the Iris dataset.
2. Split the dataset into training and testing sets.
3. Train a K-Nearest Neighbors (KNN) classifier.
4. Make predictions on the test data.
5. Evaluate the model using accuracy.

**Result**

The trained model successfully predicts the species of iris flowers and achieves a high accuracy on the test dataset.

**Task-4: Email Spam Detection with Machine Learning**

**Overview**

This project implements a **Email Spam Detection** system using **Machine Learning**. It classifies messages as **Spam** or **Ham (Not Spam)** by analyzing the text content of each message.

**Objective**

To build a machine learning model that accurately identifies whether an email or SMS message is spam or not.

**Technologies Used**

* Python
* Pandas
* Scikit-learn

**Dataset**

The project uses a spam message dataset containing two columns:

* **Label** – Indicates whether the message is `ham` or `spam`.
* **Message** – The text content of the message.

**Steps**

1. Load the dataset.
2. Preprocess the data.
3. Convert text into numerical features using **CountVectorizer**.
4. Split the dataset into training and testing sets.
5. Train a **Multinomial Naive Bayes** classifier.
6. Predict whether a message is Spam or Ham.
7. Evaluate the model using accuracy.

**Result**

The trained model successfully classifies messages as **Spam** or **Ham** and achieves high accuracy on the test dataset.

**Task-5: Sales Prediction using python**

**Overview**

This project implements a **Sales Prediction** model using **Linear Regression** in Python. The model predicts product sales based on advertising expenditures across different media channels.

**Objective**

To build a machine learning model that predicts sales using advertising budgets for:

* TV
* Radio
* Newspaper

**Technologies Used**

* Python
* Pandas
* Scikit-learn
* Matplotlib

**Dataset**

The project uses an **Advertising** dataset containing the following features:

* **TV** – Advertising budget spent on TV.
* **Radio** – Advertising budget spent on Radio.
* **Newspaper** – Advertising budget spent on Newspapers.
* **Sales** – Product sales (Target Variable).

**Steps**

1. Load the dataset.
2. Check for missing values.
3. Split the dataset into training and testing sets.
4. Train a **Linear Regression** model.
5. Predict sales for the test data.
6. Evaluate the model using **Mean Absolute Error (MAE)** and **Mean Squared Error (MSE)**.
7. Visualize the actual and predicted sales using a line graph.

**Result**

The trained model successfully predicts sales based on advertising budgets. Model performance is evaluated using MAE and MSE, and the results are visualized by comparing actual and predicted sales.
